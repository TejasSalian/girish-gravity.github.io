Overview Page Help

This page is the primary navigation page for the Queensland Infrastructure Pipeline. Visitors can narrow their search for information using the Asset Class and Regions options on the left. Click on the bubbles in the pipeline to view more detailed information about each proposal and project.

List Page Help

This page lists the proposals and projects for the Asset Classes and Regions selected on the Overview page. Visitors can sort this list by a number of attributes, search the list for specific projects, and select a project to view.

Details Page Help

Selecting a project to view from the list takes you to this detailed project information page. Use the list on the left of the screen to switch between different projects.
